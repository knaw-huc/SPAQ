SPAQ Recorder

Information

SPAQ Web Recorder is based on web-dictaphone but is already heavily mutated to make up for some weird browser incompatibilities

Other Differences:

  • Download recorded files
  • Delete recorded files
  • Added an server for storage and conversion
  • Wrapped it up in Docker
  • Made the oscilloscope workable in different browsers

It's built using getUserMedia and the MediaRecorder API, which provides an relatively easier way to capture Media streams.
